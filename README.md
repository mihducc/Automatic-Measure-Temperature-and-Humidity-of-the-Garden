## Electronic sensors and components that are required for the project
1. STM32F103C8T6 
2. Soil moisture sensor
3. IC LM2576 
4. LCD 16x02 
5. IC LM35 
6. Mini water pump 12V
7. Relay 12V 
8. Multi-purpose transformer
